## About Labaid Insuretech Customer Portal

The Labaid Insuretech B2B Dashboard is a web application built with Next.js for managing insurance policies, claims, and customer interactions. Key features include:

- **Employee Management**: View, update, and renew insurance policies seamlessly.
- **Department Processing**: Submit and track insurance claims with real-time status updates.
- **Insurance Plans**: Access personalized dashboards for account overview, payment history, and support tickets.
- **Purchase Orders**: Integrated user authentication with role-based access control.
- **Billings**: Optimized for desktop and mobile devices using modern UI frameworks.
- **Settings**: Settings

## Project Setup Process

### Prerequisites

- Node.js (version 18 or higher)
- npm, yarn, pnpm, or bun package manager
- Git for version control

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Sujon-Ahmed/labaidinsuretech-b2b-dashboard.git
   cd labaidinsuretech-b2b-dashboard
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   ```

3. Set up environment variables:
   - Copy `.env.example` to `.env.local` and fill in the required API keys, database URLs, and other configurations.

4. Run the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
