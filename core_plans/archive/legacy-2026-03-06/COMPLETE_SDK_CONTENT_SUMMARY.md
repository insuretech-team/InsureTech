## 7. RUN_API_PIPELINE.PS1 CONTENT

PowerShell script for OpenAPI generation pipeline with 16 steps:
0. Proto file generation
1. Cleanup old files
2. Code generation (proto → schemas)
3. OpenAPI assembly
4. Validation
5. Coverage analysis
6. Documentation generation
7. Swagger UI generation
8. ReDoc generation
9. Schema visualization
10. Docker validation (optional)
11. HTML report generation
12. Test SDKs generation
13. Apidog sync (optional)
14. Code quality checks
15. HTTP server startup

Parameters:
- `-SkipCleanup`: Skip cleanup step
- `-SkipValidation`: Skip validation step
- `-SkipDocs`: Skip documentation generation
- `-Fast`: Skip validation and serve immediately
- `-ServerPort`: Server port (default 8080)

Server endpoints:
- http://localhost:PORT/ (redirects to docs)
- http://localhost:PORT/docs/swagger.html
- http://localhost:PORT/docs/redoc.html
- http://localhost:PORT/docs/index.html (schema visualizer)
- http://localhost:PORT/openapi.yaml
