# BRD V3.7 - Modular Documentation Structure

## Overview

This directory contains the modularized source files for BRD V3.7. The final document `BRDV3.7.md` is generated by merging all components using the `merge_brd_v3_7.py` script.

## Directory Structure

```
BRD/
├── README.md
├── merge_brd_v3_7.py
├── BRDV3.7.md                  (generated)
├── sections/
│   ├── 00_header.md
│   ├── 01_business_overview.md
│   ├── 02_scope.md
│   ├── 03_stakeholders.md
│   ├── 04_user_stories.md
│   ├── 05_business_processes.md
│   ├── 06_requirements_summary.md
│   ├── 13_project_planning.md

```

## Quick Start

### 1. Edit a Section

```bash
notepad sections/04_user_stories.md
```

### 2. Regenerate Final Document

```bash
python merge_brd_v3_7.py
```
